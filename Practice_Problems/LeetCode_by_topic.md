# LeetCode Practice — by topic

Curated, service-company-relevant problems mapped to your roadmap. Mostly **Easy** (that's
the realistic level for TCS/Infosys/Wipro/Accenture/Cognizant rounds), with a few **Medium**
stretches marked ⭐. Tick them off as you go.

> Workflow: solve on LeetCode in Python, then drop your accepted code into the matching
> `DayXX` folder or a scratch file. Aim to dry-run one example on paper first (your habit fix).
